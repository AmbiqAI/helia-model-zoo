# Audio Models

This domain collects speech and audio-adjacent models, including the MLPerf Tiny keyword spotting and streaming wake word references plus larger standalone speech models.

| Model | Family | Task | I/O | Docs |
| --- | --- | --- | --- | --- |
| KWS Reference | MLPerf Tiny | keyword spotting | 1 input / 1 output | [kws_ref](mlperf-tiny/kws_ref/README.md) |
| Streaming Wake Word | MLPerf Tiny | wake word detection | 1 input / 1 output | [strm_ww](mlperf-tiny/strm_ww/README.md) |
| RNNoise | standalone | speech denoising | 4 inputs / 5 outputs | [rnnoise](rnnoise/README.md) |
| Wav2Letter | standalone | speech recognition | 1 input / 1 output | [wav2letter](wav2letter/README.md) |
| DFNet2 | standalone | speech enhancement | 3 inputs / 5 outputs | [dfnet2](dfnet2/README.md) |
| GTCRN | standalone | speech enhancement | 1 input / 1 output | [gtcrn](gtcrn/README.md) |
